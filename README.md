# DmmD-Web / Bunsvr

## Synopsis

A small template repository for all of my Bun web servers.

## Installation

```
# Clone the repository
git clone https://github.com/dmmd-web/bunsvr proj-name
cd proj-name

# Install packages
bun i

# Starts server
bun .
```

## Configuration

```.env
# Server port
PORT=3000
```
